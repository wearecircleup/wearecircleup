# Herramientas para Crear Material Educativo de Calidad en Circle Up

### Herramientas de Diseño y Creación de Contenido

- [**Canva:**](https://www.canva.com/) Plataforma intuitiva con plantillas para crear presentaciones, infografías, carteles y otros materiales visuales.
- [**Genially:**](https://genial.ly/) Herramienta para crear presentaciones interactivas, infografías, juegos, cuestionarios y otros contenidos con animaciones y elementos multimedia.
- [**Visme:**](https://www.visme.co/) Plataforma para crear presentaciones, infografías, gráficos y otros contenidos visuales con funciones de colaboración en tiempo real.
- [**Google Slides:**](https://www.google.com/slides/about/) Herramienta gratuita de Google para crear presentaciones con plantillas, imágenes, videos y elementos interactivos.

### Herramientas de Creación de Videos

- [**Lumen5:**](https://lumen5.com/) Herramienta que convierte automáticamente artículos de blog o textos en videos, agregando imágenes, música y efectos visuales.
- [**Animoto:**](https://animoto.com/) Plataforma para crear videos animados a partir de fotos, videos y música, con plantillas personalizables y efectos de transición.
- [**Biteable:**](https://biteable.com/) Plataforma para crear videos explicativos y presentaciones animadas, con una biblioteca de plantillas, personajes y elementos gráficos.

### Herramientas de Contenido Interactivo

- [**Edpuzzle:**](https://edpuzzle.com/) Plataforma que permite convertir videos educativos en lecciones interactivas, agregando preguntas, comentarios y notas.
- [**Quizlet:**](https://quizlet.com/) Herramienta para crear tarjetas de estudio interactivas, juegos y cuestionarios para repasar y reforzar el aprendizaje.
- [**Kahoot!:**](https://kahoot.com/) Plataforma de juegos educativos que permite crear cuestionarios y competiciones en tiempo real para reforzar el aprendizaje.
- [**Mentimeter:**](https://www.mentimeter.com/) Plataforma para crear presentaciones interactivas con encuestas, cuestionarios, nubes de palabras y otras herramientas de participación del público.

### Herramientas de Colaboración

- [**Google Workspace for Education:**](https://workspace.google.com/) Suite de herramientas gratuitas que incluye documentos, presentaciones, hojas de cálculo y formularios, ideales para colaborar en la creación de materiales educativos.
- [**Padlet:**](https://es.padlet.com/) Herramienta para crear murales virtuales colaborativos donde los participantes pueden compartir ideas, recursos y comentarios.
- [**Miro:**](https://miro.com/) Pizarra colaborativa online para realizar lluvias de ideas, mapas mentales, diagramas y otras actividades en equipo.
