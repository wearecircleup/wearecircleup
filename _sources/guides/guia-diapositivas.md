# Guía Rápida de Diseño de Diapositivas para Circle Up

### Fuente

- Poppins (disponible en [Google Fonts](https://fonts.google.com/specimen/Poppins))

### Tamaños

- <span class="h-word">Título:</span>48-60 puntos (negrita)
- <span class="h-word">Subtítulo:</span>32-36 puntos (negrita o regular)
- <span class="h-word">Párrafo:</span>24-28 puntos (regular)

### Colores

- Usa la paleta de colores de Circle Up.
- Fondo claro, texto oscuro.
- Resalta elementos importantes con colores vibrantes.

### Imágenes

- Usa imágenes de alta calidad y relevantes.
- Herramientas: [Unsplash](https://unsplash.com/), [Pexels](https://www.pexels.com/es-es/), [Bing Image Creator](https://www.bing.com/create)

### Diseño

- <span class="h-word">Menos es más:</span>Evita texto excesivo.
- Usa la regla de los tercios para una composición equilibrada.
- Deja espacio en blanco.
- Alinea los elementos de forma coherente.

### Consejos Extra

- Revisa y edita tus diapositivas.
- Practica tu presentación.
- Pide retroalimentación.
